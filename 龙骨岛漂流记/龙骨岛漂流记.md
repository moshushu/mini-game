# 龙骨岛漂流记 - 详细设计方案

使用 JavaScript + HTML + CSS 实现的轻量级荒岛生存模拟游戏。

## 1. 游戏背景
你在一场海难中幸存，漂流到一座无人荒岛。你需要利用岛上的有限资源（木材、石头、食物等）生存下去，并寻找逃离海岛的方法。

## 2. 核心机制 (Core Mechanics)
游戏采用 **实时时间推进制**，游戏内时间自动流逝，玩家需要及时采取行动维持生存。

### 2.1 时间系统 (Time System)
- **时间比例:** 现实中 **5分钟 = 游戏内1天（24小时）**
- **每小时时长:** 现实中约 **12.5秒 = 游戏内1小时**
- **自动流逝:** 时间自动推进，玩家需及时补充资源
- **自然消耗:** 每小时自动消耗饥饿-2、水分-3、精力-1

### 2.2 生存指标 (Status Metrics)
- **生命值 (Health):** ❤️ 核心指标，归零则游戏结束。受饥饿、渴度和环境伤害影响。
- **饥饿度 (Hunger):** 🍖 随时间流逝而减少，通过食用食物恢复。
- **水分 (Thirst):** 💧 随时间流逝而减少，减少速度比饥饿度快。
- **精力 (Energy):** ⚡ 采集、建造和探索会消耗精力，必须通过休息/睡眠恢复。

### 2.3 资源与物品系统 (Items & Resources)
- **基础材料:** 木头 (Wood) 🪵、石头 (Stone) 🪨、纤维 (Fiber) 🧶。
- **食物与饮品 (可消耗):**
  - **浆果 (Berries) 🍒:** 基础食物，少量补充饥饿和水分。
  - **生鱼 (Fish) 🐟:** 补充饥饿，但可能导致健康下降（不卫生）。
  - **罐头 (Canned Food) 🥫:** 高级食物，大量补充饥饿。
  - **矿泉水 (Bottled Water) 💧:** 基础饮品，补充大量水分。
  - **可乐 (Cola) 🥤:** 稀有饮品，补充水分并提升精力。
  - **椰子 (Coconut) 🥥:** 同时补充水分和少量饥饿。
- **工具制作 (Tools):** 
  - **石斧 (Axe) 🪓:** 加速木材采集效率。
  - **石镐 (Pickaxe) ⛏️:** 加速石头采集效率。

### 2.4 探索与环境 (Environment)
- **地点系统:**
  - **海滩 (Beach) 🏝️:** 容易发现矿泉水或漂流物。
  - **森林 (Forest) 🌲:** 盛产木材和浆果。
  - **山洞 (Cave) 🕳️:** 石头资源丰富。
  - **山顶 (Mountain) 🏔️:** 容易采集到藤蔓（纤维）。
- **昼夜循环:** 18:00 后进入夜晚，夜晚休息效果更佳，但部分活动受限。
- **随机事件系统:**
  - **暴雨:** 补充水分，但可能改变天气。
  - **大风:** 恶劣天气导致生命值受损。
  - **流星:** 夜晚限定，恢复精力。
  - **资源点:** 偶遇野果丛等额外补给。

## 3. 用户界面 (UI) 设计
通过现代化的 Web 界面提供即时的图形交互反馈。

### 3.1 布局规划 (Layout)
- **顶部状态栏 (Header):** 彩色进度条显示四大生存指标，带实时数字显示（如 100/100）。
- **中间展示区 (Display):** 
  - **环境视图:** 
    - 左上角显示实时游戏时钟（如 06:00）
    - 中央显示当前地点图标、名称及天数时间信息
    - 右上角显示暂停 ⏸️ 和重新开始 🔄 按钮
  - **滚动日志:** 记录玩家的所有行为、后果及突发事件（最新消息在底部）。
- **右侧背包 (Inventory):** 网格化展示物品，显示物品图标、名称和数量，**可点击使用**消耗品。
- **底部动作面板 (Actions):** 
  - **采集:** 砍树 🪵、搜寻食物 🥩、寻找水源 💧。
  - **探索:** 🧭 切换当前地理位置。
  - **制作:** 🛠️ 弹出模态框进行工具合成。
  - **休息:** 🧘 小憩 或 😴 睡觉以恢复精力。

### 3.2 交互增强 (Visual Feedback)
- **即时提示 (Tooltip):** 鼠标悬停在状态栏时，**瞬间弹出**详细的功能文字描述。
- **层级管理:** 状态栏置于顶层（z-index: 200），确保在任何弹窗下都清晰可见。
- **动态效果:** 进度条平滑过渡，物品悬停高亮，采集动作带日志反馈。
- **游戏控制:** 
  - **暂停功能:** 点击 ⏸️ 暂停游戏，时间停止流逝，资源不再消耗。
  - **重新开始:** 点击 🔄 重置所有游戏状态，清空存档。

## 4. 技术方案 (Technical Implementation)

### 4.1 技术栈
- **HTML5:** 语义化结构，CSS 驱动的自定义 Tooltip。
- **CSS3:** Flexbox/Grid 布局，z-index 层级控制，过渡动画。
- **JavaScript (Vanilla):** 面向对象的游戏逻辑开发。

### 4.2 核心模块
- **Game Engine:** 处理时间流逝、随机事件触发及状态自减。
- **Inventory Manager:** 管理物品存取、工具判定及物品使用效果。
- **UI Controller:** 实时同步玩家属性到 DOM 元素，处理模态框交互。
- **Persistence:** 使用 `localStorage` 实现无缝的存档与读档。
- **Game Controls:** 暂停/继续、重新开始等游戏控制功能。

### 4.3 扩展路线
- **建造系统:** 消耗木材和石头建造避难所（减少夜晚生命消耗）。
- **烹饪系统:** 制作营火，将生鱼加工为熟鱼（增加回复量且无副作用）。
- **结局系统:** 制作求救信号灯或木筏，达成逃离结局。
- **难度设置:** 添加简单/普通/困难模式，调整时间流速和资源消耗。
